MCP Server

The Botersson MCP server allows Claude Desktop, Cursor, and other MCP-compatible clients to control the bot.

Add to your MCP config file:

{
  "mcpServers": {
    "botersson": {
      "command": "python",
      "args": ["-m", "bot.services.mcp_server"],
      "cwd": "/path/to/Botersson",
      "env": {
        "PYTHONPATH": "/path/to/Botersson"
      }
    }
  }
}

Default MCP config location:
macOS: ~/Library/Application Support/Claude/claude_desktop_config.json
Windows: %APPDATA%\Claude\claude_desktop_config.json

Discord Bot
Bot Commands Available:
  • /search <query> – Web search
  • /search_scope <scope> <query> – Scoped search (web/news/academic/patents/github/government)
  • /patent <query> – Patent search
  • /article <query> – Article search
  • /arxiv <query> – arXiv search
  • /analyze_article <url> [topic] – Analyze article
  • /analyze_patent <url> [topic] – Analyze patent
  • /classify <topic> <text> – Classify text
  • /ask <question> – Ask AI
  • /db_summary – Database summary
  • /chart [bar|pie|line] [topic] – Generate chart
  • /export_excel – Download Excel
  • /add_theme <type> <name> – Add theme

Set your token in .env → DISCORD_TOKEN

macOS Ventura Setup Guide
1. Install Python & Ollama
brew install python@3.11
brew install ollama
ollama pull llama3.2
2. Install dependencies
cd /path/to/Botersson
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
3. Configure & Run
cp .env.example .env
# Edit .env with your token
ollama serve &
python run.py
All components are compatible with macOS Ventura (13.x) on both Apple Silicon and Intel (Hackintosh).
